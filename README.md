# GetSoluce2

This repository contains the GetSoluce2 Laravel application. It provides a backend API and optional frontend for managing client solutions.

## Usage
1. Install dependencies with `composer install` and `npm install`.
2. Copy `.env.example` to `.env` and update your environment settings.
3. Generate the application key using `php artisan key:generate`.
4. Run migrations with `php artisan migrate`.
5. Start the development server using `php artisan serve`.
